# Automated-Employee-Onboarding-Off-boarding-System
Automated-Employee-Onboarding-Off-boarding-System



The Automated Employee Onboarding & Offboarding project in ServiceNow streamlines the entire employee lifecycle through automation. It replaces manual HR, IT, and security tasks with a unified workflow built using Flow Designer. When HR raises a request, the system automatically triggers manager approvals, creates departmental tasks, tracks SLAs, and updates the status in a custom Employee Lifecycle table. Role-based ACLs ensure data security, while UI policies make catalog forms dynamic and user-friendly. Testing confirmed seamless approvals, task completion, and notifications. The result is a secure, efficient, and fully traceable onboarding process, ready for future integration with HR and IT systems.
